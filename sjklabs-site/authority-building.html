How it runs

The six-month minimum is there because authority does not compound in one burst. The first months build the rhythm. The later months build recall.

Months 02–03

Months 04–05

Month 06

What has to be true before this starts

The narrative needs to be defined. One decision-maker needs to own approvals. The founder or named spokesperson needs to show up. And if the site needs technical work, the client needs a dev resource or specialist partner ready to implement it separately.

If those conditions are missing, the right move is usually earlier-stage strategy work first, not forcing the retainer to carry the wrong job.
